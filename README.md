# Student Course Fee Analysis (Power BI)

This repository contains a Power BI project focused on analyzing student course fees. The dashboard provides data-driven insights into fee structures, revenue patterns, and student enrollment trends.

## 🚀 Project Overview
The objective of this project is to transform raw educational data into actionable business intelligence. By visualizing course fee structures, this dashboard helps stakeholders understand revenue distribution, identify high-performing courses, and optimize financial planning.

## 📊 Key Features
*   **Revenue Analysis:** Breakdown of total fee collections by course and department.
*   **Enrollment Trends:** Visual representation of student intake over specific time periods.
*   **Fee Distribution:** Comparison of fee structures across different academic programs.
*   **Interactive Filters:** Use slicers to drill down into data by course, semester, or academic year.

## 🛠 Tools & Technologies
*   **Data Visualization:** Microsoft Power BI
*   **Data Cleaning/Prep:** Power Query Editor
*   **Analysis:** DAX (Data Analysis Expressions) for calculated measures

## 📁 How to Use
1.  **Download the `.pbix` file:** Clone or download the repository to access the Power BI file.
2.  **Open in Power BI Desktop:** Ensure you have the latest version of [Power BI Desktop](https://powerbi.microsoft.com/desktop/) installed.
3.  **Data Refresh:** If the data source is local, update the file paths in Power Query settings to point to your local dataset.
